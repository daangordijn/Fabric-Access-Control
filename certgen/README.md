### `certgen`
```                                                   
Usage: certgen [options] [command]

Options:
  -V, --version    Output the version number
  -h, --help       Display help for command

Commands:
  inspect <path>   Inspect the specified certificate
  register <name>  Register a new client certificate
  help [command]   Display help for command
```
